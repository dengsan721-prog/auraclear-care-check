# 音小鲸关爱自检 V6：紧凑视觉中心与立体产品展示设计

## 首页

- 顶部只保留简洁“听”字标志与 `CARE CHECK`，不重复显示品牌名称。
- “听见关爱”作为首屏最大文字，位于视觉中心附近。
- 内容组不采用大面积空白居中；标志、标题、痛点、说明、按钮和信任说明以紧凑间距均匀排列。
- 文案固定为：
  - 主标题：“听见关爱”；
  - 痛点：“父母总让你再说一遍？”；
  - 说明：“用 10 个生活场景，了解最近父母的日常听力状态。”；
  - 按钮：“开始检测”；
  - 信任说明：“约3分钟 · 不采集资料 · 不是医学诊断”。

## 检测页

- 顺序固定为：返回与题号、进度条、题目、真实作答提示、四个选项。
- 提示为：“请按近一段时间真实发生的情况选择，不按‘应该怎样’或想象中的情况作答。”
- 题目、提示和选项连续排列，压缩上下空白；390 × 844 单题无需滚动。

## 结果页行动层级

- 结果数据、专家解读和返回修改继续保留。
- 第一行动为“看看怎么和父母说”，突出沟通优先。
- 第二行动为“想听清一点，去体验”，进入产品试听页。
- “今天可以和父母一起测”继续保留为陪测引导，但视觉层级低于沟通主行动。

## 沟通页

- 导航下方不保留大段空白，标题紧接顶部。
- 删除“复制沟通话术”按钮和 `copyTalk()` 相关代码。
- 保留核心开场话术与七种父母反馈应对。
- 页面末尾只保留“想听清一点，去体验”主按钮。
- 顶部可返回结果页，产品页可返回沟通页。

## 产品页

- 导航下方不保留大段空白，产品标题和说明紧接顶部。
- 保持暖白背景和五个图标式特点。
- 特点说明缩短为一句核心价值，降低纵向滚动长度。
- 主图舞台采用浅色径向渐变；产品主体适度放大。
- 使用产品图容器裁切、清晰对比度、边缘高光、前后光影和椭圆接触阴影形成更强立体感。
- 主图允许轻微旋转与位移，但不得使用 `mix-blend-mode` 或模糊产品主体。
- 电话区继续居中，无复制电话号码按钮。

## 全流程合理性修正

- 首页不重复展示品牌名。
- 结果页明确“先沟通、再体验”，避免两个同权按钮造成选择困难。
- 沟通页和产品页压缩顶部空间。
- 产品特点标题和说明保持可扫读，不堆叠参数或夸大效果。
- 页面返回关系、10 题答案、分数结果和陪测引导均保留。

## 验收标准

- 首页文案与顺序完全符合本设计，主标题明显放大且无大面积留白。
- 真实作答提示位于题目之后、选项之前。
- 结果页同时存在“看看怎么和父母说”和“想听清一点，去体验”，且沟通优先。
- 沟通页无复制话术功能，顶部紧凑。
- 产品页顶部紧凑，耳机主体清晰并具有明显接触阴影、边缘高光和空间层次。
- 五个特点仍存在且说明更短。
- 页面无二维码、无表单、无外部图片依赖。
- GitHub Pages 公网返回 200 并展示 V6 内容。
