# 音小鲸关爱自检 V3 苹果式精简设计

日期：2026-07-14  
状态：待用户书面审核

## 1. 设计目标

将 V2 从“内容完整”进一步压缩为“每屏只有一个结论和一个动作”。采用用户选定的 B 方案“温润象牙”视觉，以及 2 号结果方案“分值光谱”。

核心路径：

1. 首页直击痛点。
2. 六题紧凑检测。
3. 分值光谱与对应解读。
4. 添加今晚回家自测提醒。
5. 沟通方法与话术。
6. 父母愿意了解后进入产品与试听。

## 2. 视觉系统

- 主背景：温润象牙白 `#F5F2EC`。
- 主文字：深墨绿 `#173D32`。
- 强调色：雾金 `#B18A50`。
- 中性色：Apple 风冷灰 `#6E6E73`、浅灰 `#E5E5EA`。
- 字体：系统无衬线为主，使用粗细、字号和留白建立层级。
- 圆角克制使用，主要按钮使用 22–28px 胶囊圆角。
- 不使用多张卡片堆叠、装饰性声波、复杂图标和大段说明。
- 主要内容宽度不超过 680px，移动端一屏完成一个任务。

## 3. 首页

首页不展示产品图、型号、痛点列表和产品入口。

主标题：

> 最近，父母总让你再说一遍？  
> 也许不是没听见，而是听清越来越费力。

说明：

> 用 6 个生活场景，快速了解日常聆听状态。

主按钮：`开始检测`

信任说明：`约2分钟 · 不采集资料 · 不是医学诊断`

## 4. 检测页

- 顶部只显示返回、细进度线和 `1/6`。
- 中部显示一个问题，字号 28–34px。
- 下方显示四个横向选项按钮。
- 删除大号题目编号、重复的场景标签和说明段落。
- 手机常见高度内不需要滚动即可完成选择。
- 选择后自动进入下一题；最后一题完成后直接进入结果。

## 5. 结果页

### 5.1 分值光谱

使用大号数字，例如 `63`，下方只显示分层名称：

- 0–29：继续观察。
- 30–59：值得关注。
- 60–100：建议行动。

数字下方显示一条三段式光谱，并用圆点标记用户所在位置。结果页不使用圆环、仪表盘或多张数据卡片。

只显示得分最高的两个生活场景标签。

### 5.2 按分值对应解释

0–29：

> 暂未发现明显困扰。继续观察电视音量、重复沟通和多人聊天的变化即可。

30–59：

> 部分场景已经需要更多提醒或重复。可以找一个具体场景，温和问问父母最近是否听得比较累。

60–100：

> 多个生活场景已经受到影响。建议尽早安排正规听力检查，再了解适合的日常聆听支持方式。

### 5.3 专家解读

专家内容根据分值变化，每次只显示一段：

- 0–29：偶尔听漏受注意力、距离和环境影响较大，暂时以持续观察为主。
- 30–59：安静环境尚可、复杂环境费力时，问题可能不只是声音大小，还与语音分辨有关。
- 60–100：多个场景同时出现困难时，亲人可能正在依靠猜测、观察口型或提高音量补偿，应尽早接受正规检查。

固定边界：`以上为生活场景解读，不能代替专业听力检查。`

医疗提醒：`突然单侧听力变化、耳痛、眩晕或耳鸣加重，请及时就医。`

### 5.4 行动按钮

主按钮：`提醒我今晚和父母一起测`

点击后生成 `.ics` 日历文件，默认时间为当天 19:30–19:50；如果当前时间已经晚于 19:30，则生成次日 19:30。标题为“和父母一起做日常聆听自检”，备注中包含 GitHub Pages 链接。

次按钮：`现在看看怎么和父母说`

## 6. 沟通话术页

核心方法：

> 不评价听力，只描述你看到的具体场景。

默认展开主话术：

> 最近我们说话时，有几次我要重复两遍。我不是嫌你听不清，就是担心你听着比较累。我们先去体验一下，不检查、不买东西也没关系。

三个折叠问题：

- 父母说“我听得见”。
- 父母担心花钱。
- 父母暂时不愿意谈。

只允许展开一个问题，避免信息同时铺开。

按钮：

- `复制这段话`。
- `父母愿意了解，查看试听方案`。

## 7. 产品与试听页

开场：

> 不喜欢堵住耳朵，又担心操作复杂，可以先体验开放式骨传导辅听。

展示一张高质量产品大图和三个价值点：

- 开放式不入耳。
- 日常人声调控。
- 辅听与蓝牙两用。

不再展示五个长产品模块、长参数和重复边界说明。

产品边界：`是否合适，以亲人的实际佩戴和试听感受为准。`

免费试听：

- 电话：`199 9195 1867`。
- 主按钮：`拨打电话`。
- 次按钮：`复制号码`。
- 承诺：`免费预约、免费试听；不强制购买，不诱导转账。`

页面底部保留“扫码打开自检页面”二维码。

## 8. 状态与异常处理

- 检测结果、提醒和复制操作均在本机完成，不上传个人资料。
- Clipboard API 不可用时使用隐藏文本框复制。
- 日历下载失败时显示自定义提示：今晚19:30和父母一起检测。
- 产品图片加载失败时仍保留产品名称、三个价值点和试听电话。
- 用户返回修改答案后重新计算分数和专家解读。

## 9. 验收标准

1. 首页正文不超过一个标题、一句说明和一行信任信息。
2. 首页不出现产品图、型号、产品价值或购买入口。
3. 检测页在 390×844 手机视口中无需滚动即可完成选择。
4. 结果页使用大数字和三段式光谱，不出现圆形分数图。
5. 三个分值区间分别显示不同的解释和专家解读。
6. 结果只展示两个最高分场景。
7. 日历提醒时间计算符合 19:30 规则并包含页面链接。
8. 沟通页默认只显示主话术，三个补充问题使用单开折叠。
9. 产品页只展示一张大图和三个价值点。
10. 试听页没有信息表单，只显示电话、拨号、复制和一行承诺。
11. 页面保持移动端苹果式简洁层级、足够对比度和至少 48px 点击区域。
